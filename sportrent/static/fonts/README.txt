Шрифты для экспорта PDF (кириллица).

Чтобы PDF корректно отображал русский текст и на Windows, и на Linux:

1. Скачайте DejaVu Sans: https://dejavu-fonts.github.io/Download.html
   или установите пакет в системе:
   - Ubuntu/Debian: sudo apt install fonts-dejavu-core
   - тогда шрифты подхватятся из /usr/share/fonts автоматически.

2. Если используете локальную копию — поместите в эту папку файлы:
   - DejaVuSans.ttf
   - DejaVuSans-Bold.ttf

После этого экспорт в PDF будет работать с кириллицей на любой ОС.
